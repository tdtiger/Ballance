# Ballance
Unity3Dプロジェクトを何か作ってみたいと思い作成．

## 概要
画面上部から迫り続ける壁の穴にボールが落ちてしまわないよう、転がして回避していくゲーム。
仮ビルドのため、クリア条件等は特になく、失敗するまでゲームは続きます。
